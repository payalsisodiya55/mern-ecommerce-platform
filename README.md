# 🛍️ E-Commerce Web Application with Admin Dashboard (MERN Stack)


Build a small-scale E-Commerce platform using the MERN Stack (MongoDB, Express.js, React.js, Node.js)  
with dynamic product management, cart functionality, order handling, and an admin dashboard.

---

 Tech Stack Used

 Frontend:
- React.js (Vite) – UI framework for fast rendering
- React Router DOM – Page navigation
- Axios – API communication between frontend & backend


 Backend:
- Node.js + Express.js – Server and routing
- MongoDB (Mongoose) – NoSQL database
- JWT (JSON Web Token) – Authentication & Authorization
- Bcrypt.js – Password hashing for secure login
- dotenv*– To manage environment variables
